🔧 THIẾT LẬP LẦN ĐẦU (1 LẦN DUY NHẤT):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Để LUÔN LUÔN dùng được tính năng này:

1. Tìm file opera.exe:
   C:\Users\My PC\AppData\Local\Programs\Opera\opera.exe

2. Right-click → Create shortcut

3. Right-click shortcut → Properties

4. Trong "Target", thêm vào CUỐI:
   
   --remote-debugging-port=9222

5. Đổi tên shortcut: "Opera (Debugging)"

6. Kéo ra Desktop hoặc Pin to Taskbar

✅ XONG! Từ giờ LUÔN mở Opera bằng shortcut này!
