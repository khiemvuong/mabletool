✅ THUẬT TOÁN: ĐÃ TỐI ƯU HÓA
   Tốc độ cũ: ~8-12 giây
   Tốc độ mới: ~3-5 giây
   Cải thiện: 50-70% NHANH HƠN! ⚡

✅ NHẬP NỘI DUNG: KHÔNG CÒN NHẬP TAY
   Cũ: Gõ từng ký tự (type) - slow
   Mới: JavaScript injection - siêu nhanh! ⚡
   
   Ví dụ: "hello world" (11 ký tự)
   - Trước: 1100ms (11 × 100ms)
   - Sau: < 50ms
   - Tiết kiệm: 95% thời gian!

🆕 TÍNH NĂNG MỚI:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. ⚡ JAVASCRIPT INJECTION
   → Nhập text siêu nhanh (không gõ từng ký tự)
   → Tiết kiệm: 1-2 giây

2. 🚫 CHẶN RESOURCES
   → Không load images, fonts, CSS không cần
   → Tiết kiệm: 2-3 giây

3. 🎯 CUSTOM SELECTORS
   → Cung cấp CSS selector cụ thể để bỏ qua dò tìm
   → Tiết kiệm: 2-3 giây
   
   Ví dụ:
   - Search Selector: input#search
   - Result Selector: a#video-title

4. ⏭️ SKIP REFRESH
   → Bỏ qua bước F5 (khi không cần)
   → Tiết kiệm: 2-3 giây

5. 🔢 CHỌN VỊ TRÍ KẾT QUẢ
   → Click vào kết quả thứ N (không chỉ đầu tiên)
   → Chính xác hơn!
